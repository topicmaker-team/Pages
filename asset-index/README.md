## 運用メモ
- 1つの制作物 = 1エントリ。更新は新しい行を増やさず、その制作物の「更新履歴」に日付付きで追記する。
- 新規追加時は、このページの全件一覧と該当カテゴリページの両方に追記する。
- 各エントリ冒頭の `<a id="...">` がアンカー。一覧表のタイトルから飛べる。
- 設計・UIと、そこから実装されたアプリは相互リンクで紐付ける（設計側に「関連実装」、アプリ側に「設計元」を記載）。
- 件数が増えて絞り込み・検索が欲しくなったら、このフォルダのデータからGitHub Pages用のHTML一覧ページ（検索・種別フィルタ・並べ替え）を生成する案がある。
