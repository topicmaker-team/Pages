# 繰り上げ返済 最適化シミュレーター

住宅ローンの繰り上げ返済を「軽減型」「短縮型」で比較し、住宅ローン控除を加味した実質コストをシミュレーションできるツールです。

## 機能

| 機能 | 説明 |
|------|------|
| 軽減型シミュレーション | 繰り上げ後の新しい月額・月々削減額・実質利息を計算 |
| 短縮型シミュレーション | 繰り上げ後の新しい返済期間・短縮月数・実質利息を計算 |
| 比較表 | 軽減型 vs 短縮型を一覧で比較 |
| 控除詳細 | 年次ローン残高と住宅ローン控除額（軽減型・短縮型）を表示 |

## 使い方

1. `index.html` をブラウザで開く（サーバー不要）
2. 上部の入力欄に **現在残債・金利・月額返済額** を入力
3. スライダーで **繰り上げ元本額（10万〜10,000万円）** を調整
4. ボタンで確認したい内容を切り替え

## 入力項目

| 項目 | 範囲 | デフォルト |
|------|------|-----------|
| 現在残債 | 任意（万円） | 3,000万円 |
| 金利（年利） | 0.1〜8% | 0.5% |
| 月額返済額 | 20万円未満 | 8万円 |
| 繰り上げ元本 | 10万〜10,000万円 | 100万円 |

## 計算の前提

- **軽減型**：繰り上げ後の残債を、元の残り期間そのままで再計算 → 月額が下がる
- **短縮型**：繰り上げ後の残債を、元の月額そのままで再計算 → 期間が縮まる
- **住宅ローン控除**：年末残高 × 0.7%（上限35万円/年、最大13年間）で試算
- 実際の控除額は取得年・入居年・建物種別等により異なります
